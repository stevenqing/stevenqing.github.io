# Shuqing Shi (史述青) - Personal Academic Website

个人学术主页，托管在 GitHub Pages。

## 🔗 网站地址

部署后访问: `https://你的用户名.github.io`

## 📁 文件说明

- `index.html` - 网站主页
- `cv.pdf` - 简历文件
- `photo.jpg` - 头像照片 (需要自己添加)

## 🚀 部署步骤

### 1. 创建 GitHub 仓库
1. 登录 GitHub
2. 创建新仓库，名称必须是: `你的用户名.github.io`
3. 设为 Public

### 2. 上传文件
1. 点击 "Add file" → "Upload files"
2. 上传 `index.html` 和 `cv.pdf`
3. 点击 "Commit changes"

### 3. 等待部署
1-2 分钟后访问 `https://你的用户名.github.io`

## ✏️ 自定义

### 添加头像
1. 准备一张照片，命名为 `photo.jpg`
2. 上传到仓库
3. 编辑 `index.html`，找到:
```html
<!-- Replace with: <img src="photo.jpg" alt="Shuqing Shi"> -->
S
```
改成:
```html
<img src="photo.jpg" alt="Shuqing Shi">
```
